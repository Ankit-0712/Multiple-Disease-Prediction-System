# Multiple Disease Prediction System
The Multiple Disease Prediction System is a machine learning application that predicts the likelihood of various diseases such as Heart Disease, Diabetes, and Parkinson's Disease based on user-provided health metrics. The application uses trained machine learning models to provide accurate predictions and assist in early detection of these diseases.

Features
Heart Disease Prediction: Predicts the probability of heart disease using health parameters.
Diabetes Prediction: Determines the likelihood of diabetes based on medical data.
Parkinson's Disease Prediction: Analyzes symptom data to predict Parkinson's disease.
User Input Based Predictions: Accepts user input to generate disease predictions.
Streamlined and Efficient: Quick and accurate predictions using machine learning algorithms.
